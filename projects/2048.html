[2048 Game]

One of the addictive game I have ever played after subway surfer. The day I played this game I dreamed of creating my version of it. Finally after months of creative efforts I am able to finish it.

I tried to make it as generic as possible. This implementation can be configured to create many variations of the game without writing any code.

Features of this implementation

  • Can be configured to create game board of any size
  • Can be themed easily
  • Even the collision formula, (what happens when to tiles with same value collides) can be changed from the configuration.
  • Game state is saved, game can be continued even after browser reload
