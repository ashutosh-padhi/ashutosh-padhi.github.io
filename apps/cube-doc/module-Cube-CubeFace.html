Class: CubeFace

Cube~CubeFace(face, tilefn)

new CubeFace(face, tilefn)

Face of the cube. The face of the cube is made of a square grid of tiles. It is a 2d structure. This 2d place can be sliced horizontally module:Cube~CubeFace#sliceY or sliced vertically module:Cube~CubeFace#sliceX.

Parameters:
Name Type Description
face Array.<Array.<string>>

2d array representing the face of the cube, where each tile can be of any color data, f,b,u,d,l,r.
tilefn module:Cube~tilefn

Callback to execute before creating of each tile.
Properties:
Name Type Description
face Array.<Array.<module:Cube~CubeTile>>

An array representing the face of the cube.
size number

Size of the face, it is same as size of the cube.
