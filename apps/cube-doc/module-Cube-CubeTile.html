Cube~CubeTile(color, position, data, rotation)

new CubeTile(color, position, data, rotation)

A single cube tile.

Parameters:
Name Type Description
color string

Color of the tile
position module:Cube~Position

Postion of the tile in current CubeFace
data Object

Any data that needes to be stored per tile
rotation number

Angle in degree, multiples of 90. This is the rotation applied to the tile because of the cube operations. this rotation is not helpful if cube is made of flat color faces. But if cube is made of faces with images, then each tile rotation also need to be handled.
