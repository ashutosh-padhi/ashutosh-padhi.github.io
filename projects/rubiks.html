[Rubik's Cube]

A generalized implementation of Rubik's cube of any size. Either you want to play with Rubik's cube or analyze how a particular Rubik's cube algorithm works or want to create some other application with Rubik's cube, you will find this project helpful to you.

First time I saw a Rubik's cube somewhere around the age of 12. It was belongs to one of my friends. I was fascinated by seeing how all the pieces can move with respect to each other. As time passed I had learnt some algorithm to solve the cube. The day I solved my first cube, I have created the curiosity in me to know how the algorithm works. I tried finding a good simulator for doing this. Then I thought why not create another simulator.

Design

I tried to keep the cube logic and the UI rendering logic as separate as possible. So that it can be used with any js runtime to render on any ui framework if needed. There is an implementation to run the cube in console. and another implementation to run the cube in browser.

Terminal App

The cube model and the logic that manipulates the model is completely isolated from the logic that render the cube. So that the same model can be used to render the cube in some other way. One example of this can be rendering the cube in terminal.
